About Me.

Hello, I'm Waleed, a passionate Game Developer with expertise in crafting engaging 2D platformer games and exploring the exciting world of 3D development in Unity. As a skilled programmer, I excel in building intricate game systems and mechanics that bring gameplay to life. While I’m still learning Unreal Engine and honing my artistic skills, I thrive on creating immersive experiences through innovative problem-solving and system design. My Projects

Explore my recent game development projects where I showcase my creativity, skills, and passion for building immersive experiences.

Tiny Titans

This is the major project that I worked on in my final year of college with a team. It is a 3D platform adventure game, where you have to collect memory fragments to complete levels and get to the end. It has mechanics like shooting, platforming, and dodging.

Kazuki's Blade vs Ogres

This is a 2D platformer game that I made using Unity. I added mechanics such as player movement and jump with a combo attack system.

Zero Deaths

This is a 2D platfomer where you have to avoid obstacles and complete each room. If the player dies once the game restarts. This was my first ever unity project.

Pop Duel

This was a team project that was made for Global Game Jam. It is 2-4 player local multiplayer game where the last man standing wins. I was one of the lead programmers in this project, this game has multiple weapons with different effects.

Food VS Germs

This is a tower defense game for android. In this game you get currency by killing germs and then using this currency to get more utensils to defend your food.

Return To Origins

This is a 2D isometric mobile game which was made for Nostra Game Jam by our team. I was the developer for the movement, level mechanics and puzzle mechanics.

WiP (Work in Progress)

Built these systems from scratch to strengthen my skills in Unity, C#, and gameplay architecture. Designed the project as a self-challenge to showcase problem-solving, system design, and technical implementation capabilities.
